Nesse curso iremos aprender a utilizar o HTML5 e o CSS

HTML5

O HTML5 é uma linguagem de programação utilizada para fazer sites e páginas de internet.

CSS

O CSS não é considerado uma linguagem de programação. Ele é uma coleção de estilos.

1 FORMATAÇÃO DE TEXTO

1.1 Formatação de Título

É utilizado o h1 até o h6.

1.2 Ênfase

É utilizado strong para Negrito e em para Itálico.

1.3 Formatação de Parágrafo

p define um parágrafo.

1.4 Quebra de Linhas

br define uma quebra de linha.

1.5 Citações

blockquote para citações em blocos e q para citações curtas

1.6 Listas

ul para listas não ordenadas, ol para listas ordenadas e li para itens da lista.
